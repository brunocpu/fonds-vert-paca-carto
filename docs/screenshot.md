# Capture d'écran à fournir

Le `README.md` référence `docs/screenshot.png` mais le fichier n'a pas encore été produit.

## Capture suggérée

- **Vue** : carte affichant les 2 années (bouton `2023+2024` actif), zoom sur la région PACA entière, panneau Stats à droite visible.
- **Format** : PNG, largeur ≈ 1600 px (Retina recommandé pour le confort de lecture sur GitHub).
- **Cadrage** : inclure le bandeau jaune disclaimer (intégral, lisible) — c'est un signal fort de la posture POC.
- **État des filtres** : tous activés (vue par défaut). Optionnel : recherche `region` pour montrer les pills de portée.

## Production

1. Ouvrir https://brunocpu.github.io/fonds-vert-paca-carto/ (ou local).
2. Attendre la fin du chargement, fermer toutes les popups Leaflet.
3. Capture d'écran (Outil Capture Windows, ou `Cmd+Shift+4` sur macOS).
4. Sauvegarder en `docs/screenshot.png`.
5. Supprimer ce fichier `docs/screenshot.md` après production.
